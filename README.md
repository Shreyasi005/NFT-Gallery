# 🖼️ NFT Gallery (Soroban Smart Contract)

## 📌 Project Description
NFT Gallery is a basic decentralized application built using Soroban (Stellar Smart Contracts). It allows users to mint, store, and view NFTs on the Stellar blockchain.

This project demonstrates how NFTs can be implemented in a simple and efficient way using Rust and Soroban SDK.

## 🚀 What it does
- Users can mint NFTs by providing:
  - Owner address
  - NFT name
  - Metadata URI (image or asset link)
- Stores NFT data on-chain
- Allows retrieval of:
  - Individual NFTs by ID
  - All NFTs in the gallery

---

## ✨ Features
- 🔐 Secure minting with authentication
- 🧾 On-chain NFT storage
- 🖼️ Simple NFT gallery listing
- ⚡ Lightweight and efficient contract
- 🔎 Fetch NFT by ID
- 📚 Retrieve all NFTs

## 🛠️ Tech Stack
- Soroban SDK (Rust)
- Stellar Blockchain

---

## 📦 Functions Overview

### `mint(owner, name, uri)`
Creates a new NFT and assigns ownership.

### `get_nft(id)`
Fetches a specific NFT by its ID.

### `get_all()`
Returns a list of all NFTs stored in the contract.
